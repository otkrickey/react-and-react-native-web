# sh-fes2021

## Required

For Android:

* AndroidStudio(for android development)
* Java SE Development Kit(JDK)
* Android SDK
* Android SDK Platform
* Android Virtual Device

## Installation

```bash
yarn install
```

## Setup

```bash
yarn build
```

## Start

### React Native Project (ios,android)

```bash
yarn start mobile
```

### React Project (web)

```bash
yarn start web
```
